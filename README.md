# cheatsheet.creco.me

터미널 내에서 미리 지정해둔 문서를 불러올 수 있는 커맨드입니다. glow 를 통해 Terminal 에서 markdown View를 지원합니다.

## requirements

- homebrew
- wget

## Install

```bash
wget -q -O - https://creaticoding.github.io/cheatsheet.creco.me/install.sh | bash
```

## Update

```bash
ch update
```

## Uninstall

```bash
ch uninstall
```
